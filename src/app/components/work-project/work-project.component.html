Oh-My-Wrap

Workshop website where you can create wrapping paper with unique design

Client Private entrepreneur
role Lead developer
Team Vestbery
Problem:

Ð¡ustomers should be able to create their own wrapping paper design through a full cycle from creation to order confirmation and payment

Solution:

A web site that supports functionality of typical online store and an online tool that allows you to quickly and easily create wrapping paper design

Project description:

Website has the same functionality as online store such as registration, authorization (via login and password or social networks), basket, session state, payment acceptance etc.

Manager can check orders and download created wrapping paper designs in special format for further processing.

Also backoffice allows to flexibly configure each site element (texts, images, settings etc.)

Wrapping paper creation online tool allows customers to upload images from their devices or social networks.

Then easily order, crop uploaded images if needed. Uploaded images and their state are restored after page reload.

On this project i was responsible for:
  • Initial backend and frontend architecture
  • Integration with major bank (Sberbank) to use their acquiring system to handle payments on the website
  • Integration with CMS Umbraco on the basis of which backoffice is built.
  • User authorization and registration via login and password or via social networks
  • Basket management and session restoration
  • All the functionality of online tool of creating wrapping paper design
