Telegram birthday notification bot

Telegram bot that will help you not to forget about birthdays of important people

Goal:

Create telegram bot that can notify users about birthdays

Main functionality:
  • Import birthdays list
  • View imported birthdays
  • Clear imported birthdays
  • Set notification delay in days
Code technologies I got involved with while working on this project:
  • C#
  • Hangfire
  • Telegram API
